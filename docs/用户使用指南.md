## 一、快速入门

### 1.1 什么是元数据治理平台？

本平台帮助您：
- **看清数据全貌** - 了解公司 Tableau 中有哪些数据库、数据表、数据源、工作簿
- **追溯数据来源** - 查看任意报表字段的数据来自哪张表、经过哪些加工
- **发现治理问题** - 找出重复的指标、无人使用的报表、缺失描述的字段
- **评估数据健康** - 通过健康度评分了解整体数据治理水平

## 二、首页概览

首页是您了解数据治理全局状况的起点。

### 2.1 治理健康度

页面顶部的圆环图显示整体健康评分（0-100分）：

| 分数范围 | 状态 | 含义 |
|----------|------|------|
| 80-100 | 🟢 健康良好 | 数据治理状况优秀，持续保持 |
| 60-79 | 🟡 需要关注 | 存在一些问题，建议优化 |
| 0-59 | 🔴 亟需治理 | 问题较多，需要重点处理 |

### 2.2 五维治理评分

健康度由五个维度组成：

| 维度 | 评估内容 | 如何提升 |
|------|----------|----------|
| **完整性** | 字段/表是否有描述 | 补充缺失的描述信息 |
| **规范性** | 数据源是否已认证 | 对重要数据源进行认证 |
| **时效性** | 数据源更新频率 | 确保 Extract 定期刷新 |
| **一致性** | 是否有重复指标 | 统一重复的计算公式 |
| **有效性** | 资产是否被使用 | 清理无人使用的资产 |

### 2.3 资产全景

七个资产类型的统计卡片，点击可跳转到对应页面：

- **数据库** - Tableau 连接的数据库
- **数据表** - 物理数据表
- **数据源** - Tableau 已发布数据源
- **工作簿** - Tableau 工作簿
- **视图** - 仪表板和工作表
- **原始字段** - 来自数据表的字段
- **计算字段** - 用户创建的计算公式

### 2.4 行动中心

按优先级列出需要处理的治理问题：

| 优先级 | 问题类型 | 处理建议 |
|--------|----------|----------|
| 🔴 高 | 重复公式 | 统一指标定义，删除重复 |
| 🔴 高 | 停更数据源 | 检查刷新任务，或标记废弃 |
| 🟡 中 | 缺失字段描述 | 补充业务含义说明 |
| 🟡 中 | 孤立表 | 确认是否需要，或清理 |
| ⚪ 低 | 缺失指标描述 | 补充计算逻辑说明 |
| ⚪ 低 | 未认证数据源 | 完成认证审核 |

点击问题项可直接跳转到对应的分析页面。

### 2.5 血缘桑基图

展示数据从物理层到展示层的完整流向：

```
数据库 → 数据表 → 数据源 → 原生字段 → 工作簿 → 视图
                         → 计算字段 ↗
```

**颜色含义**：
- 🔵 蓝色：正常资产
- 🟣 紫色：嵌入式资产
- 🔴 红色：孤立/僵尸资产
- 🟠 橙色：Custom SQL

点击节点可跳转到对应页面。

## 四、字段管理

### 4.1 原始字段页面

**路径**: 左侧菜单 → 原始字段

管理来自数据表的原始字段。

**标签页切换**：

| 标签 | 说明 |
|------|------|
| 原始字段目录 | 去重后的标准字段列表 |
| 无描述字段 | 缺失描述的字段 |
| 孤立字段 | 无视图引用的字段 |
| 热门字段 | 被引用最多的字段 |

**什么是去重？**
同一个物理表的同一列，可能在多个数据源中被使用。系统会将它们聚合为一个"标准字段"，避免重复计数。

**卡片信息**：
- 字段名称
- 角色：维度 / 度量
- 数据类型
- 所属表
- 实例数（在多少个数据源中存在）
- 热度（被多少个视图引用）

---

### 4.2 计算字段页面

**路径**: 左侧菜单 → 计算字段

管理用户创建的计算公式/指标。

**标签页切换**：

| 标签 | 说明 |
|------|------|
| 计算字段目录 | 去重后的标准指标列表 |
| 同名不同公式 | 名称相同但公式不同的指标 ⚠ |
| 同公式不同名称 | 公式相同但名称不同的指标 |
| 高复杂度 | 复杂嵌套的计算公式 |
| 未使用指标 | 无视图引用的指标 |

**重点关注：同名不同公式**
这表示同一个指标名称在不同地方有不同的计算逻辑，可能导致数据口径不一致！

**复杂度等级**：
| 等级 | 分数范围 | 说明 |
|------|----------|------|
| 低 | 0-1 | 简单重命名，如 `[字段名]` |
| 中 | 2-7 | 中等计算，如 `SUM([Sales])` |
| 高 | >7 | 复杂嵌套，如多层 IF/CASE/LOD |

## 五、治理分析

### 5.1 如何发现问题资产？

每个模块都有"治理分析"相关的标签页：

| 模块 | 治理分析内容 |
|------|-------------|
| 数据表 | 孤立表（无下游引用） |
| 数据源 | 未认证分析、孤立数据源 |
| 工作簿 | 空工作簿（无视图） |
| 视图 | 零访问视图、热门视图 |
| 原始字段 | 无描述、孤立字段、热门字段 |
| 计算字段 | 重复指标、高复杂度、未使用 |

### 5.2 详情面板

点击任意资产卡片，右侧会弹出详情面板。

**通用标签页**：

| 标签 | 内容 |
|------|------|
| 概览 | 基本信息、属性、描述 |
| 血缘 | 上下游关系图 |

**资产特定标签页**：

以数据源为例：
- 上游表：使用了哪些物理表
- 字段：包含哪些字段（按表分组）
- 计算字段：包含哪些指标
- 下游工作簿：被哪些工作簿使用

### 5.3 血缘追溯

在详情面板的"血缘"标签页，可以看到该资产的上下游关系图。

**使用场景**：
- 想知道某个报表的数据来自哪张表？→ 查看工作簿/视图的血缘
- 想知道某张表被哪些报表使用？→ 查看表的血缘
- 想知道修改某个字段会影响哪些报表？→ 查看字段的血缘

### 5.4 跳转到 Tableau

在工作簿、视图等资产的详情面板中，如果显示了 Tableau Server 链接，点击可直接跳转到 Tableau Server 中查看原始内容。

## 六、搜索与导航

### 6.1 全局搜索

**快捷键**: `Cmd+K` (Mac) / `Ctrl+K` (Windows)

在顶部搜索框输入关键词，可搜索：
- 数据库、数据表
- 数据源、工作簿、视图
- 字段、计算字段

输入后按回车跳转到搜索结果页面。

### 6.2 页面内搜索

每个列表页面都有搜索框，可按名称搜索当前类型的资产。

### 6.3 筛选器

列表页面顶部提供筛选器：

**筛选器类型**：
- **Facet 筛选**：显示各选项的数量，支持多选
- **下拉筛选**：点击展开选择

**常用筛选维度**：
| 页面 | 可筛选维度 |
|------|-----------|
| 数据库 | 连接类型 |
| 数据表 | Schema、数据库 |
| 数据源 | 认证状态、项目 |
| 工作簿 | 项目 |
| 视图 | 视图类型、工作簿 |
| 字段 | 角色、去重方式 |
| 计算字段 | 角色、聚合状态 |

### 6.4 排序

点击排序按钮可切换排序方式：
- 点击一次：降序 ↓
- 再点一次：升序 ↑
- 再点一次：取消排序

### 6.5 分页

列表页面底部有分页控件：
- 左右翻页按钮
- 页码跳转
- 每页显示条数切换（20/50/100）

### 6.6 导航历史

在详情面板中点击关联资产会打开新的详情：
- 面板顶部显示导航路径（面包屑）
- 点击路径可返回之前的资产
- 点击返回按钮 ← 可逐级返回

## 八、常见问题

### Q1: 数据多久更新一次？

数据通过后台同步任务从 Tableau Server 采集。同步频率由管理员配置。
您可以在首页概览查看"最后同步时间"。

### Q2: 为什么有些资产显示"嵌入式"？

嵌入式资产是直接嵌入工作簿中的资源，而非独立发布到 Tableau Server。
例如：嵌入式数据源、嵌入式表。

这类资产只在特定工作簿中使用，无法被其他工作簿共享。

### Q3: 什么是"孤立"资产？

孤立资产指没有上下游关联的资源：
- 孤立表：没有数据源使用
- 孤立数据源：没有工作簿使用
- 孤立字段：没有视图引用

孤立资产可能是废弃的旧资源，建议定期清理。

### Q4: 为什么同一个字段有多个"实例"？

同一个物理表的同一列，可能被多个数据源引用。
系统会显示"实例数"表示这个字段在多少个地方被使用。

在"原始字段目录"中，系统会将这些实例聚合为一个标准字段。

### Q5: 如何判断一个指标是否有问题？

关注以下几点：
1. **同名不同公式** - 可能存在口径不一致
2. **高复杂度** - 可能难以维护
3. **无描述** - 他人难以理解
4. **未使用** - 可能是废弃指标

### Q6: 如何查看某个报表使用了哪些数据？

1. 找到对应的工作簿或视图
2. 点击卡片打开详情面板
3. 查看"上游表"或"使用的数据源"标签页
4. 或查看"血缘"标签页的可视化图

### Q7: 如何知道修改数据会影响哪些报表？

1. 找到要修改的表或字段
2. 点击卡片打开详情面板
3. 查看"下游工作簿"或"使用视图"标签页
4. 或查看"血缘"标签页，关注下游方向

### Q8: 详情面板中的"帮助"按钮是什么？

点击详情面板右上角的帮助图标 (?) 可打开"介绍弹窗"，
查看当前资产类型和标签页的详细说明与图解。

### Q9: 如何导出数据？

当前版本暂不支持数据导出功能。
如需导出，请联系系统管理员。

### Q10: 发现数据不准确怎么办？

数据来源于 Tableau Server，如发现不准确：
1. 首先确认 Tableau Server 中的原始数据
2. 检查最后同步时间，可能数据尚未同步
3. 联系管理员触发数据重新同步
